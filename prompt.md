[역할]
너는 20년 차 퀀트 개발자이자 Python 전문가야.
나를 위해 미국 주식과 매크로 지표를 한눈에 보여주는 'Streamlit 대시보드'를 만들어줘.

[목표]
매일 아침 내가 보유한 종목의 '기술적 위험 신호'와 '매크로 상황'을 30초 만에 파악하고 싶어.

[기능 요구사항]
1. 라이브러리: `streamlit`, `yfinance`, `pandas_ta`, `plotly` 사용.
2. 대상 종목 (Default): IONQ, PLTR, FIG(Figma), NVDA, TSLA, SKM(SK Telecom)
3. 핵심 지표 (기술적 분석):
   - 각 종목의 현재가, 전일 대비 등락률(%)
   - RSI(14): 30 미만이면 '🔴 과매도(Buying Chance)', 70 초과면 '🔵 과열(Warning)'로 강조 표시.
   - 최근 5일간의 주가 추세(Sparkline) 또는 미니 차트.
4. 매크로 지표 (시장 분위기):
   - 미국 10년물 국채 금리(^TNX)와 비트코인(BTC-USD)의 현재가 및 등락률을 최상단에 요약(Metric)으로 표시.
5. UI/UX:
   - 다크 모드에 어울리는 깔끔한 디자인.
   - 모바일에서도 보기 편하게 '카드' 형태나 직관적인 데이터프레임 사용.

[코드 구조]
- 하나의 파일(`app.py`)로 실행 가능하게 작성.
- 주석을 달아서 각 부분이 어떤 역할을 하는지 설명할 것 (내가 나중에 수정할 수 있게).
- 에러 처리: 티커를 못 불러와도 앱이 멈추지 않게 `try-except` 처리 필수.

[출력]
위 요구사항을 완벽하게 구현한 Python 전체 코드를 작성해 줘.
